# Extremes

Most programming languages can be divided roughly along one or more of these extremes:

1. Manual Memory Management (i.e. C, Zig, Odin)
1. Object-Oriented (i.e. Smalltalk, Java)
1. Functional  (i.e. Haskell, Gleam, Lisp)
1. Logic  (i.e. Prolog)
