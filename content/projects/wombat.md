# Wombat

Wombat is a [WASM](https://webassembly.org) [virtual machine](https://wikipedia.org/wiki/Virtual_machine). Or at least, it will be. Someday.

Right now it is a learning project. I currently have the parser done, and most of the validation.

## Zig

Wombat is written in [Zig](https://ziglang.org). Zig is an imperative, lower level language similar to C but with updated syntax and ergonomics. I've really enjoyed using Zig.
